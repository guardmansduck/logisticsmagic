| Secret Name                   | Description                                          |
| ----------------------------- | ---------------------------------------------------- |
| `APP_STORE_CONNECT_API_KEY`   | Base64-encoded `.p8` API key content                 |
| `APP_STORE_CONNECT_ISSUER_ID` | Issuer ID from App Store Connect                     |
| `APP_STORE_CONNECT_KEY_ID`    | Key ID from App Store Connect                        |
| `APP_IDENTIFIER`              | Your app bundle identifier (e.g., com.example.MyApp) |
| `TEAM_ID`                     | Apple Developer Team ID                              |
| `FASTLANE_PASSWORD`           | Optional if using Apple ID login instead of API key  |
